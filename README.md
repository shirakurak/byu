# byu

### 環境
- Node.js: 14.7.5
  - Tools for Native Module：チェック
- Vue CLI をインストール
  - `npm install -g @vue/cli`
- プロジェクト作成
  - `vue create byu-journal`
    - プリセット：`Default(Vue 3 Preview) ([Vue 3] babel, eslint)`
- プロジェクト実行
  ```
  $ cd byu-journal
  $ npm run serve
  ```

### 目的
Vue.jsを用いたサイトを作成することを目的とする。

docs配下においてあるのは暫定の資料。
